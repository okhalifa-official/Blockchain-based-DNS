# Contributing to Blockchain-based DNS

Thank you for your interest in contributing to **Blockchain-based DNS**
Contributions are welcome and appreciated, whether they are bug reports, feature requests, documentation improvements, or code contributions.


## Project Status

> ⚠️ **This project is under active development and is experimental.**  
> It is currently deployed on the **Sepolia testnet only** and is **not production-ready**.


## Ways to Contribute

You can contribute by:

- Reporting bugs or unexpected behavior
- Suggesting new features or improvements
- Improving documentation or README clarity
- Enhancing smart contract security
- Optimizing resolver performance or caching
- Improving frontend UI/UX
- Writing tests


## Reporting Issues

Before opening an issue:
1. Check existing issues to avoid duplicates
2. Clearly describe the problem
3. Include:
   - Expected behavior
   - Actual behavior
   - Steps to reproduce
   - Relevant logs or screenshots (if applicable)
