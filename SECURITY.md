# Security Policy

## ⚠️ Project Status
> This project is **experimental** and **under active development**.  
> It is deployed on the **Sepolia testnet only** and is **not production-ready**.
>
>❗ **Do NOT use real cryptocurrency or real private keys.**

---

## Reporting Vulnerabilities

If you find a security issue:
- Open a **GitHub Issue** labeled `security`
- Include clear steps to reproduce and potential impact

Please avoid public disclosure of critical vulnerabilities before review.

---

## Scope

Relevant security issues include:
- Smart contract logic or access control flaws
- Resolver or cache vulnerabilities
- Unsafe Web3 or wallet interactions
- Misconfiguration or secret exposure
